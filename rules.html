Rules

Morphscape is in early testing - these are the baseline rules while things stabilize.

Respect other players

No harassment, hate speech, or targeted abuse in public chat or towards staff.

No exploiting or cheating

Client modification, macroing, and abusing bugs for personal gain aren't allowed. Report bugs instead - it helps the server.

No real-world trading

Buying or selling in-game items, currency, or accounts for real money is not permitted.

One account per player

Unless a game mode explicitly allows it (e.g. alting for Ironman content), stick to a single main account.

Staff decisions are final

If you disagree with a moderation action, raise it calmly and directly - don't argue in public chat.

These rules will grow

As the server matures, expect more specific rules around PvP, clans, and the economy. Check back here.

How to Play

A quick primer beyond the homepage basics.

Commands

Type ::commands in game for the full list. ::help covers general questions, and ::redeem claims store purchases.

Getting geared

Morphscape includes end-game best-in-slot gear (Scythe of Vitur, Torva, Masori, Tumeken's Shadow, and more) alongside standard OSRS content.

Something wrong?

This is an actively developed test server - if something looks broken, it probably is. Let staff know what you found.
