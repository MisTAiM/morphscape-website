Getting Started

Everything you need to jump in.

1. Download the Client

Grab the Morphscape client and run it - no account setup needed to get to the login screen.

2. Create a Character

Log in with any username and password to create your account on first login.

3. Play

Type ::help in game for a full list of commands, and ::redeem to claim any store purchases.
